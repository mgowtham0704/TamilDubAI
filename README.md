# 🎬 TamilDubAI

AI-powered automatic video dubbing platform.

TamilDubAI converts videos into Tamil speech using AI technologies:
- Speech Recognition
- Machine Translation
- Tamil Text-to-Speech
- Automated Audio-Video Merging

## 🚀 Features

✅ Video upload and processing  
✅ Automatic speech transcription  
✅ English → Tamil translation  
✅ Tamil AI voice generation  
✅ Male/Female voice selection  
✅ Timestamp-based dubbing  
✅ Streamlit web interface  
✅ Download final dubbed video

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/TamilDubAI.git

Move into project folder:

cd TamilDubAI

Create virtual environment:

python -m venv .venv

Activate environment:

Windows:

.venv\Scripts\activate

Install requirements:

pip install -r requirements.txt
▶️ Run Application

Start TamilDubAI:

streamlit run app.py
