# 🎬 TamilDubAI

AI-powered movie dubbing system that converts English video content into Tamil audio using Artificial Intelligence.

## 🚀 Overview

TamilDubAI automatically:

1. Extracts speech from a video
2. Converts speech to text using Whisper AI
3. Translates the script into Tamil
4. Generates natural Tamil speech using Neural TTS
5. Synchronizes audio with original timestamps
6. Merges the Tamil audio with the video
